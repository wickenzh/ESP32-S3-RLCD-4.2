# ESP32-S3 RLCD 4.2 天气时钟

这是一个基于 **ESP32-S3** 和 **4.2 英寸 RLCD 屏幕** 的低功耗天气时钟固件项目。

本仓库是公开源码镜像，只包含时钟固件工程本体。项目中的私有部署脚本、本地自动化、密钥、发布产物、上位机工程和设备私有服务地址不会同步到这里。

## 项目定位

这个项目面向一台长期摆放使用的桌面天气时钟设备。它不是单纯的时间显示器，而是把本地传感器、天气服务、低功耗屏幕、音频提醒和 OTA 更新整合在一起的嵌入式终端。

项目设计目标：

- 常驻显示：屏幕长期保持信息可读。
- 低功耗运行：只在需要时开启 Wi-Fi、音频和高频刷新。
- 本地可用：支持离线模式和 RTC 时间保持。
- 可维护：页面、网络、传感器、音频、OTA 等逻辑按模块拆分。
- 可扩展：预留资源分区，方便后续通过上位机写入自定义图片资源。

## 主要功能

- 多页面 RLCD 时钟界面，支持按键切换和页面排序。
- 天气时钟：显示时间、日期、天气、天气预警、本地温湿度、电池状态、Wi-Fi/声音状态。
- 图片时钟：显示本地图片、分钟级时间和每日文字。
- 翻页时钟：使用高对比数字时钟样式，秒级局部刷新。
- 温度历史：记录并显示 24 小时温湿度趋势。
- 日历：显示当月日历、农历、节日和今日高亮。
- 天气看板：显示城市天气、空气质量、湿度、风力、日出日落和未来天气。
- 设置菜单：支持网络、声音、显示、系统等分类设置。
- 页面开关与排序：可关闭部分页面，并自定义工作页顺序。
- 离线模式：可通过手动设置时间进入离线工作状态，停止网络同步。
- OTA 更新：支持通过远程固件清单检查并更新固件。
- 低电量模式：电量过低时进入极简显示并限制高功耗行为。

## 页面体系

设备当前的工作页以“可关闭、可排序、按需刷新”为核心：

- 天气时钟适合日常主界面显示，包含时间、天气、预警、状态图标和整天进度。
- 翻页时钟强调大数字和秒级局部刷新，适合只看时间的场景。
- 图片时钟用于展示本地图片和每日文字。
- 温度历史保留本地 24 小时温湿度趋势。
- 日历显示当月日期、农历和节日信息。
- 天气看板展示更完整的城市天气、空气质量、风力、日出日落和未来天气。

页面切换、页面启用状态和页面顺序均保存在设备本地。天气时钟作为兜底页面始终可用，避免用户关闭所有页面后无法显示工作界面。

## 数据与联网

设备通过 Wi-Fi 获取外部数据，但不会让 Wi-Fi 长期开启：

- 天气数据来自 QWeather。
- 每日文字来自公开文字 API。
- 时间同步通过 NTP。
- 固件更新通过 OTA manifest 获取版本信息。

天气、每日文字和 OTA 都会受到离线模式、低电量模式、配网状态和 OTA 状态保护，避免在不合适的时机触发网络请求。离线模式开启后，设备会停止所有非配网网络行为，只依赖 RTC 和本地缓存继续显示。

## 低功耗策略

RLCD 屏幕适合长时间显示，因此固件尽量只刷新变化区域：

- 秒级页面只刷新必要的数字或进度区域。
- 低刷新页面按分钟、小时、日期变化或传感器变化刷新。
- 天气和每日文字按页面可见性触发，不在后台无意义刷新。
- 电池采样、温湿度采样和 Wi-Fi 启停根据运行状态调整频率。
- 低电量时进入极简显示，限制高功耗行为。

## 硬件与固件组成

- 主控：ESP32-S3。
- 屏幕：4.2 英寸 RLCD。
- 传感器：本地温湿度采样用于页面显示和历史记录。
- 音频：用于整点提醒、设置试听等提示音。
- 网络：Wi-Fi 用于配网、天气同步、每日文字、OTA 更新和网络检测。
- 存储：NVS 保存 Wi-Fi、API Key、页面设置、声音设置、历史采样等必要数据。

## 目录结构

- `main/`：固件主程序和业务模块。
- `components/`：板级支持、屏幕、音频、资源等组件。
- `partitions.csv`：ESP32-S3 分区表。
- `CMakeLists.txt`：ESP-IDF 工程入口。
- `sdkconfig.defaults`：默认工程配置。

## OTA 与资源

设备端通过固件内置的 OTA 清单地址获取最新版本信息，再下载对应 app 固件。完整刷写固件、发布脚本、Cloudflare R2 / Gitea / 私有自动化配置不在本公开仓库中提供。

项目还预留了自定义资源分区，用于后续通过上位机写入图片时钟和天气时钟相关素材。内置素材会作为兜底存在，避免用户资源损坏导致页面空白。
